# DEFENSE_PREPARATION.md

## Common Defense Questions and Suggested Answers

| Question | Recommended Answer |
| --- | --- |
| Why not just use Google Drive? | Google Drive mainly stores and shares files. ReNote adds academic repository organization, visibility control, access requests, AI summaries, summary history, citations, search, and trust labels for academic resources. |
| Why not Google Scholar? | Google Scholar helps users find published research. ReNote helps users manage uploaded learning materials, course files, summaries, and institutional resources. |
| Why not ResearchGate? | ResearchGate is researcher-centered. ReNote is learning-resource-centered for students, faculty, and institutions. |
| Why include AI summarization? | AI summarization directly addresses information overload by helping users understand long academic files faster. |
| Why save summary history? | Users need to revisit generated summaries without opening the original file every time. This makes the AI feature more useful for studying. |
| Why use public/restricted/private visibility? | Academic resources need different access levels. Some can be public, some require approval, and some should remain private. |
| Why include access requests? | Restricted repositories need a controlled way for users to request permission instead of giving open access to everyone. |
| Why include repository views in MVP? | Repository view tracking supports dashboard statistics and recent activity, helping users see which resources are being accessed. |
| Why are Collections not MVP? | Collections are useful, but the core MVP already covers repository organization, files, search, access, and AI summaries. Collections can be added after the core system is stable. |
| Why is Archive Second Wave? | Archive improves long-term organization, but the MVP can work without it. The main priority is repository, file, access, search, and AI summary functionality. |
| Why is the advanced citation generator Future? | Basic summary citation metadata is enough for MVP. Full APA/MLA/IEEE formatting adds complexity and can be developed later. |
| How does ReNote reduce information overload? | It centralizes academic resources, organizes them into repositories/folders, controls access, allows search, and generates AI summaries with history for faster review. |

## Defense Positioning

ReNote should be presented as a focused academic resource management system, not a general file storage clone. Its main value is combining organized repositories, controlled access, search, AI summaries, and summary history to make learning resources easier to manage and review.

## MVP Boundary Explanation

The MVP focuses on the features needed to prove the system concept. Second Wave and Future features are intentionally separated to avoid scope creep and to keep development realistic for the team timeline.
